# Iteration 06 - Lead Analytics

## Goal

Переработать аналитику под лиды, стадии, активности и работу сотрудников.

## Scope

- Добавить backend расчет KPI по лидам.
- Добавить dashboard по стадиям воронки.
- Добавить аналитику просроченных активностей.
- Добавить аналитику утренних планов и вечерних отчетов.
- Добавить метрики по сотрудникам.
- Добавить фильтры по периоду, стадии, сотруднику, источнику и ответственному.
- Обновить frontend dashboard.

## Out of scope

- Не делать BI-конструктор.
- Не удалять старые отчеты до решения о миграции.
- Не делать прогнозирование.

## Definition of Done

- Dashboard показывает лиды по стадиям.
- Dashboard показывает просроченные активности.
- Dashboard показывает сотрудников без вечернего отчета.
- Dashboard показывает среднее время лида в стадии.
- Фильтры работают на backend и frontend.
- Старый dashboard либо сохранен отдельно, либо аккуратно заменен.

## Verification

```bash
cd backend
npm run build
```

```bash
cd frontend
npm run build
```

Ручная проверка:

- открыть dashboard;
- применить фильтр по сотруднику;
- применить фильтр по стадии;
- сверить числа с тестовыми данными.
