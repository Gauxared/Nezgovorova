# Project Scope — Информационная система аналитической отчетности

## Назначение

Разработать fullstack web-приложение для аналитической отчетности ООО «7 Красных линий».

Система должна заменить разрозненную подготовку отчетов в файлах и таблицах единой информационной средой: централизованное хранение, проверка данных, расчет показателей, визуализация, отчеты, экспорт, импорт и разграничение доступа.

## Проблемы предметной области

- данные о клиентах, проектах, задачах и финансах хранятся в разных источниках;
- отчеты формируются вручную;
- есть риск дублей, ошибок и разных версий файлов;
- справочники и статусы не унифицированы;
- сложно проверить происхождение показателя;
- финансовые и персональные данные требуют защиты;
- руководству нужна актуальная аналитическая панель.

## Цель системы

Создать систему, которая обеспечивает:

1. хранение клиентов, проектов, задач, финансовых записей и справочников;
2. CRUD через web-интерфейс;
3. расчет аналитических показателей;
4. dashboard с KPI и графиками;
5. отчеты и экспорт;
6. импорт CSV;
7. роли и права доступа;
8. audit log.

## Роли

- `admin` — полный доступ, пользователи, справочники, журнал;
- `director` — аналитика и отчеты;
- `project_manager` — проекты и задачи;
- `client_manager` — клиенты;
- `finance` — финансовые показатели;
- `analyst` — аналитика, отчеты, проверка данных.

## Модули

- Auth and roles;
- Users;
- Clients;
- Projects;
- Tasks;
- Finance;
- References;
- Dashboard;
- Reports;
- Import;
- Export;
- Audit log;
- Admin panel.

## Минимум для защиты

На защите нужно показать: вход, dashboard, создание клиента/проекта/задачи/финансовой записи, пересчет аналитики, отчет, экспорт, импорт, audit log и разграничение доступа.
